# Exploratory Data Analysis (EDA) Project

## Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on customer sales data to identify patterns, trends, and relationships between variables.

The project uses statistical summaries and visualizations to generate meaningful business insights.

---

# Dataset Name
Customer_Sales_EDA_Dataset.csv

---

# Technologies Used

- Python
- Pandas
- Matplotlib

---

# Features

- Load and explore dataset
- Generate statistical summaries
- Identify correlations
- Analyze purchase behavior
- Create visualizations

---

# Visualizations Included

- Bar Chart
- Histogram
- Scatter Plot
- Correlation Matrix
